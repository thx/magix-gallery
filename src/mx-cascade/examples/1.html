使用场景：

返回选中的叶子节点数据selected（value值），item（完整数据对象）
默认点击展开子项
HTML Code
{{!text1}}  
<mx-cascade class="w200"
    selected="{{=selected}}"
    list="{{@list}}"
    mx-change="select()"></mx-cascade>
