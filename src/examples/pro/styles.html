以开关组件示例如何覆盖组件样式
组件本身展示为：
自定义开关样式：
HTML Code
{{!text1}}  
<div class="outer">
    <mx-switch state="true"/>   
</div>
CSS Code
{{!text2}}  
.outer{
    [ref="@../../mx-switch/index.less:switch"] {
        height: 8px;
        border-radius: 4px;
        
        &[ref="@../../mx-switch/index.less:on"] {
            background-color: #b6eff0;

            [ref="@../../mx-switch/index.less:switch-icon"] {
                background-color: #14c9ce;
            }
    
        }
    }
}
