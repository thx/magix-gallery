以下示例：
手动打开loading进度条
进度条存在时依然可以操作页面
HTML Code
{{!text1}}  
<mx-btn brand="true" mx-click="show()" content="打开loading"/>  

<mx-btn class="ml20" mx-click="hide()" content="关闭loading"/>
JS Code
{{!text2}}  
let Magix = require('magix');
let Loading = require('@../../mx-loading/index'); //loading组件
let $ = require('$');

module.exports = Magix.View.extend({
    tmpl: '@index.html',
    mixins: [Loading],
    render() {
        this.updater.digest();
    },
    'show<click>'(e){
        // showLoading(closeMask)
        // closeMask不需要禁止操作
        this.showLoading(true);
    },
    'hide<click>'(e){
        this.hideLoading();
    }
});
