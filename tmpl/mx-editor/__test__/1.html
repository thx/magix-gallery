以下示例：
部分文案可编辑，配置展示模板tmpl
hover元素显示编辑小笔，默认不显示编辑笔
回车和失去焦点都可触发回调，只触发一次
当前内容： {{=content}}（要求必填且不小于200）
HTML Code
{{!text1}}  
<mx-editor
    tmpl="<span>${content}</span><span class='mx-tag ml5'><span class='mx-tag-name'>打标</span></span>"
    content="{{=content}}"
    rules="{{@rules}}"
    mx-change="change()" />
JS Code
{{!text2}}  
let Magix = require('magix');
module.exports = Magix.View.extend({
    tmpl: '@index.html',
    render() {
        this.updater.digest({
            content: 2000,
            rules: {
                required: true,
                min: [200, '不小于200']
            }
        });
    },
    'change<edit>' (e) {
        // editText编辑后的文案
        this.updater.digest({
            content: e.editText
        })
    }
});
