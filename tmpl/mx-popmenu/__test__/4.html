以下示例： 右键显示菜单，不配置content而是使用<mx-popmenu></mx-popmenu>标签包裹内容，则完全可自定义控制样式
右键显示菜单 当前选中： {{if selected}}操作{{=selected}}{{/if}}
HTML Code
{{!text1}}  
<mx-popmenu 
    class="color-brand cursor-pointer"
    trigger-type="contextmenu"
    list="{{@[{
        value: 1,
        text: '操作1'
    }, {
        value: 2,
        text: '操作2'
    }, {
        value: 3,
        text: '操作3'
    }]}}">右键显示菜单</mx-popmenu>
