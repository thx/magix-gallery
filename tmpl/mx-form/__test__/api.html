可配参数 说明 类型 默认值
refresh 数据变化是否需要更新view，调用view.digest boolean false
currency 金额
currency: true
currency: [true, 自定义提示]
url 合法链接
url: true
url: [true, 自定义提示]
english 英文
english: true
english: [true, 自定义提示]
email 邮箱校验
email: true
email: [true, '自定义提示（可选）', ['domain（指定邮箱，可选）', ...domains]]
chinese 中文
chinese: true
chinese: [true, 自定义提示]
mobile 手机号码
mobile: true
mobile: [true, 自定义提示]
landline 固定电话
landline: true
landline: [true, 自定义提示]
phone 手机号码 or 固定电话
phone: true
phone: [true, 自定义提示]
number 是否为数字
number: true
number: [true, '自定义提示']
pattern 正则
pattern: ^[a-zA-Z0-9_\u4e00-\u9fa5]+$
pattern: ['^[a-zA-Z0-9_\u4e00-\u9fa5]+$', '自定义错误提示']
required 必填
required: true
required: [true, '必选']
trim 不能有空格
trim: true
trim: [true, '不能包含空格']
int 整数
int: true
int: [true, '自定义提示']
posint 正整数
posint: true
posint: [true, '自定义提示']
negint 负整数
negint: true
negint: [true, '自定义提示']
length 字个数范围
length: [2, 8]
minlength
最少多少个字
minlength: 2
minlength: [2, 自定义提示文案]
maxlength
最多多少个字
maxlength: 8
maxlength: [8, 自定义提示文案]
blength 字符个数范围，一个汉字两个字符
blength: [2, 8]
bminlength 最少多少个字符，一个汉字两个字符
bminlength: 2
bminlength: [2, 自定义提示文案]
bmaxlength 最多多少个字符，一个汉字两个字符
bmaxlength: 8
bmaxlength: [8, 自定义提示文案]
unequalto
与某些节点内容不一致
unequalto: 'id1,id2'
unequalto: ['id1,id2', 自定义提示]
equalto 与某个节点内容一致
equalto: id
equalto: [id, 自定义提示]
inranges
数字范围内异常：命中范围的属于异常，范围之外的正确
inranges: [
['(,2)', '<2'],
['[2, 3]', '2 <= x <= 3'],
['(3, 5]', '3 < x <= 5'],
['(5,)', '>5']
]
range
数字范围外，包含边界，range: [a,b]， a <= x <= b
range: [2, 8, 自定义提示文案（可选）]
rangeborder 数字范围外，不包含边界，rangeborder: [a,b]， a < x < b
rangeborder: [2, 8, 自定义提示文案（可选）]
min 大于等于某个数字，min: a，a <= x
min: 2
min: [2, 自定义提示文案]
minborder 大于某个数字，minborder: a，a < x
minborder: 2
minborder: [2, 自定义提示文案]
max 小于等于某个数字，max: b，x <= b
max: 8
max: [8, 自定义提示文案]
maxborder 小于某个数字，maxborder: b，x < b
maxborder: 8
maxborder: [8, 自定义提示文案]
union
当前校验配置多个校验规则时，是交的逻辑
如果需要按照或的逻辑则可使用此配置
union: {
min: [2, 自定义提示文案]
}
json 是否为合法json
json: true
json: [true, '自定义提示']
placement 信息展示位置，可配置right，bottom string bottom
style 信息展现样式，可选纯文案（text），盒状样式（box），icon样式（icon） string text
warn
警告类信息，对象
必然是在valid=true的前提下才会有，以上所有配置项均可配置
显示为黄色警告提示
warn: {
    style: 'text',
    min: [2, 自定义提示文案]
}
pass
通过类信息，对象
必然是在valid=true的前提下才会有，以上所有配置项均可配置
显示为绿色通过提示
warn: {
    style: 'text',
    min: [2, 自定义提示文案]
}
highlight
品牌色高亮强调信息，对象
必然是在valid=true的前提下才会有，以上所有配置项均可配置
显示为品牌色通过提示
warn: {
    style: 'text',
    min: [2, 自定义提示文案]
}
