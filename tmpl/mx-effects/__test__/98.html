以下示例：
支持配置text-key，value-key，title-key，sub-key
HTML Code
{{!text1}}  
<mx-effects.tag 
    list="{{@list}}" 
    text-key="name"
    value-key="id"
    title-key="desc"
    sub-key="list"
    mx-delete="delete()"/>
JS Code
{{!text2}} 
