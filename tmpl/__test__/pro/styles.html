magix3会对给class加前缀，支持用[ref='@path:className']覆盖样式。但是强烈建议不要直接修改组件，防止版本更新的时候被覆盖，
当业务逻辑比较定制的时候，建议写本地组件。查看关于开发本地组件的约定
以开关组件示例如何覆盖组件样式
组件本身展示为：
自定义开关样式：
HTML Code
{{!text1}}  
<div class="outer">
    <mx-switch state="true"/>   
</div>
CSS Code
{{!text2}}  
.outer {
    [ref="@../../mx-switch/index.less:switch"] {
        height: 8px;
        border-radius: 4px;

        &[ref="@../../mx-switch/index.less:open"] {
            [ref="@../../mx-switch/index.less:switch-icon"] {
                background-color: #14c9ce;
            }
        }
    }
}
