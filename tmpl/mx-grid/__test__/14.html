以下示例：
部分场景，mx-grid.body内为一个自定义view，view内有部分筛选项需要定位到卡片头部右侧
在 mx-grid.body 内使用标签 mx-grid.bfilter 包裹内容，则可定位到卡片右上角，mx-grid.body需要设置为position: relative;
HTML Code
{{!text1}}  
<mx-grid>
    <mx-grid.title content="标题" tip="提示信息" />
    <mx-vframe src="@./child" style="position: relative;"/>
</mx-grid>


<!-- child.html 实现 -->
<mx-grid.bfilter>
    <mx-calendar.datepicker class="w200 mr10"/>
    <mx-dropdown class="w200" list="{{@[1,2,3]}}" />
</mx-grid.bfilter>
<mx-grid.body>内容信息</mx-grid.body>
