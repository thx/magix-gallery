使用 <mx-grid> 标签；等同于直接用 class="grid"
其中标题部分，标题+提示信息固定向左浮动，标题上其他内容请自行控制左右浮动
 内容区域
HTML Code
{{!text1}}  
<mx-grid>
    <mx-grid.title content="标题" tip="提示信息">
        <div class="fl">
            <mx-calendar.datepicker class="w200"/>
        </div>

        <div class="fr">
            <mx-calendar.datepicker class="w200 mr10"/>
            <mx-dropdown class="w200" list="{{@[1,2,3]}}" />
        </div>
    </mx-grid.title>
    <mx-grid.body>
        内容区域
    </mx-grid.body>
</mx-grid>
