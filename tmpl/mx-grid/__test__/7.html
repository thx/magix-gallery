使用 <mx-grid> 标签；也可直接用 class grid 样式
其中标题部分，标题+提示信息固定向左浮动，标题上其他内容请自行控制左右浮动
 test
HTML Code
{{!text1}}  
<mx-grid>
    <mx-grid.title content="标题" tip="提示信息">
        <div class="fl">
            <mx-calendar.datepicker class="w200 mr10"/>
        </div>

        <div class="fr">
            <mx-calendar.datepicker class="w200 mr10"/>

            <mx-dropdown class="w200" 
                list="{{@[1,2,3]}}">
            </mx-dropdown>
        </div>
    </mx-grid.title>
    <mx-grid.body>test</mx-grid.body>
</mx-grid>
