说明：
popover默认使用span标签生成
支持使用 tag="div" 之类的自定义生成标签
以下示例：
使用a标签生成，文案居中对齐
示例： 打开外链 
HTML Code
{{!text1}}  
<mx-popover
    tag="a" 
    href="https://www.taobao.com/"
    rel="noopener noreferrer"
    target="_blank"
    content="用 a 标签生成，宽度400，文字居中对齐"
    width="400"
    text-align="center">
    打开外链<i class="mc-iconfont">&#xe7aa;</i>
</mx-popover>
