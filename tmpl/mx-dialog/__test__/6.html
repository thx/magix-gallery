展示优先级： 外部自定义属性 > 浮层内属性 > 默认属性
浮层本身写法
{{!text3}}  
let Magix = require('magix');

module.exports = Magix.View.extend({
    tmpl: '@inner.html',
    init(e) {
        this.viewOptions = e;
    },
    render() {
        this.updater.digest();
    },
    'cancel<click>'(event) {
        event.preventDefault();
        let viewOptions = this.viewOptions;
        if (viewOptions.dialog) {
            viewOptions.dialog.close();
        }
    }
}, {
    dialogOptions: {
        top: 0,
        width: 800
    }
});
