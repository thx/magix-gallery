位置修正：
指定高度的浮层，dialog打开浮层时，会相对屏幕可视范围进行修正
以下示例在按钮下方打开一个400 * 400的浮层，当默认位置超出可视范围时，组件会自动进行调整
浮层 HTML Code
{{!text1}}  
<mx-btn 
    brand="true" 
    content="打开浮层"
    mx-click="open()" />
浮层 JS Code
{{!text2}}  
let Magix = require('magix');

module.exports = Magix.View.extend({
    tmpl: '@index.html',
    render() {
        this.updater.digest();
    },
    'open<click>'(event) {
        this.mxDialog('@./demo3', {}, {
            width: 400,
            height: 400,
            target: $(event.target),
        });
    }
});
