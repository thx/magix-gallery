以下示例：
以下tab切换显示为时间轴样式
支持部分选项配置为disabled
需要渐变色则配置color和color-gradient，其中color-gradient为较深的颜色，选中项文案显示为color-gradient
不需要渐变则只配置color即可
HTML Code
{{!text1}}  
<mx-tabs.pipeline 
    color="#FD93B0"
    color-gradient="#FF0036"
    selected="2"
    list="{{@[{
        value: 1,
        text: '第一个时段',
        tips: '04.27 - 04.30'
    }, {
        value: 2,
        text: '第二个时段',
        tips: '05.01 - 05.08'
    }, {
        value: 3,
        text: '第三个时段',
        tips: '05.09 - 05.13',
        icon: 'https://img.alicdn.com/tfs/TB13e.TioY1gK0jSZFCXXcwqXXa-28-34.png'
    }, {
        value: 4,
        text: '第四个时段',
        tips: '05.14 - 05.23'
    }]}}"/>
部分禁用HTML Code
