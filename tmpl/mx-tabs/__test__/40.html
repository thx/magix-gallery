以下示例：
支持配置count-map显示对应tab的计数
map中明确返回对应数值的才显示，若没有配置不会补0处理
HTML Code
{{!text1}}  
<mx-tabs 
    list="{{@[{
        value: 1,
        text: '模块1',
    }, {
        value: 2,
        text: '模块2',
        disabled: true,
    }, {
        value: 3,
        text: '模块3'
    }, {
        value: 4,
        text: '模块4'
    }]}}" 
    count-map="{{@{
        1: 100,
        2: 10,
        3: 0
    }}}"
    selected="{{:selected{refresh:true}}}"/>
