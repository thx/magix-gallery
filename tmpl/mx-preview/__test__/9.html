以下示例： 缩略图（图片）和预览图（iframe展示）不一样，直接使用preview-data
HTML Code
{{!text1}}  
<mx-preview 
    border="true"
    type="image"
    url="//img.alicdn.com/tfscom/TB10l9lbgZC2uNjSZFnXXaxZpXa.png"
    width="400"
    height="125"
    max-width="100"
    max-height="100"
    preview-data="{{@{
        type: 'iframe',
        url: '//a1.alicdn.com/creation/html/2017/06/13/creation-2480Wzuixl78VWkc-10131895.html',
        width: 300,
        height: 250,
        scale: 0.5
    }}}"/>
