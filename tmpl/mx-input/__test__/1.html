以下示例：
mx-input包装了动效，快捷删除，搜索框样式，输入计数等功能，其他与原生input基本无差异
普通输入框： 当前输入： {{=value1}}
带一键删除的输入框： 当前输入： {{=value2}}
HTML Code
{{!text1}}  
<mx-input placeholder="请输入整数" value="{{:value1{refresh:true,int:true}}}" />

<mx-input placeholder="请输入整数" value="{{:value2{refresh:true,int:true}}}" show-delete="true"/>
