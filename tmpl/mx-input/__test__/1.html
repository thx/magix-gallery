以下示例：
mx-input包装了动效，快捷删除，搜索框，密码框，输入计数等功能
对透出事件进行统一处理，仅在数据改变时抛出change事件
普通输入框
当前输入： {{=value1}}
一键删除输入框
当前输入： {{=value2}}
HTML Code
{{!text1}}  
<!-- 普通输入框 -->
<mx-input class="w200" placeholder="请输入整数" value="{{:value{refresh:true}}}" />


<!-- 配置suffix（后缀）的保留关键词delte -->
<mx-input class="w200" suffix="delete" placeholder="请输入整数" value="{{:value2{refresh:true}}}" />
