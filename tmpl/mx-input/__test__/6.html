说明：
分类型搜索，类型下拉框默认两个字符宽度，可自定义search-width（类型下拉框宽度）
设置search-value表示选中的搜索类型
搜索类型(search-value)：{{=searchValue1}} 搜索关键词(value)：{{=value1}}
搜索类型(search-value)：{{=searchValue2}} 搜索关键词(value)：{{=value2}}
HTML Code
{{!text1}}  
<mx-input class="fl w280"
    search-list="{{@[{
        text: '计划',
        value: 'campaign'
    }, {
        text: '单元',
        value: 'adgroup'
    }]}}"
    search-value="{{:searchValue{refresh:true}}}"
    prefix="search"
    placeholder="请输入"  
    value="{{:value{refresh:true}}}"/>
