支持配置prefix设置输入框前缀，前缀仅做内容类型表意提示，无交互行为
prefix可配置一些保留关键词，设置常见输入框类型：search（搜索框），money（金额输入），user（用户输入），其余情况下原样输出

以下示例：
支持配置suffix设置输入框后缀，后缀可放置一些有交互行为的操作icon，或者纯提示类的语义表达
suffix保留关键词：search（强调搜索框），delete（一键移除，操作点），password（密码框，可操作切换是否显示），其余情况下原样输出（纯展示）
强调搜索框
一键移除
密码框
语义提示
HTML Code
{{!text1}}  
<!-- 【前缀】搜索框 -->
<mx-input class="w200" prefix="search" placeholder="请输入"  value="{{=value}}" />

<!-- 【前缀】金额输入 -->
<mx-input class="w200" prefix="money" placeholder="请输入" value="{{=value}}"  />

<!-- 【前缀】用户输入 -->
<mx-input class="w200" prefix="user" placeholder="请输入" value="{{=value}}" />

<!-- 【前缀】原样输出 -->
<mx-input class="w200" prefix="自定义前缀" placeholder="请输入" value="{{=value}}" />

<!-- 【后缀】强调搜索框-->
<mx-input class="w200" suffix="search" placeholder="请输入"  value="{{=value}}"/>

<!-- 【后缀】一键移除 -->
<mx-input class="w200" suffix="delete" placeholder="请输入"  value="{{=value}}"/>

<!-- 【后缀】密码框，点击切换是否显示 -->
<mx-input class="w200" suffix="password" placeholder="请输入" value="{{=value}}"  />

<!-- 【后缀】原样输出，语义提示 -->
<mx-input class="w200" suffix="元" placeholder="请输入"  value="{{=value}}"/>
