根据可选范围对默认值进行修正
1. 未配置默认选中项且有min时，默认取min和今天中的较大值
2. 未配置默认选中项且有max时，默认取max和今天中的较小值
HTML Code
{{!text1}}  
<mx-calendar.datepicker class="w200"
    min="{{=yesterday}}"/>

<mx-calendar.datepicker class="w200"
    min="{{=tomorrow}}"/>

<mx-calendar.datepicker class="w200"
    max="{{=yesterday}}"/>

<mx-calendar.datepicker class="w200"
    max="{{=tomorrow}}"/>
