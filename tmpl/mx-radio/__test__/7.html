使用说明：
mx-radio支持配置text由组件来包装文案
对于配置了text文案的mx-radio，会用label包裹input[type="radio"]，即<mx-radio name="radios" text="文案" value="1">，最终生成的代码如下：
<label>
    <input type="radio" name="radios" value="1" />文案
</label>
而对于未配置text文案的mx-radio，则只会有input[type="radio"]，即<mx-radio name="radios" value="1">，最终生成的代码如下：
<input type="radio" name="radios" value="1" />
以下示例为带文案，打标，提示的常用场景：
文案： {{each list as item}} {{/each}} 当前选中：{{=selected1}}
文案禁用： {{each list as item}} {{/each}}
文案提示： {{each list as item}} {{/each}} 当前选中：{{=selected3}}
文案提示禁用： {{each list as item}} {{/each}}
文案提示： {{each list as item}} {{/each}} 当前选中：{{=selected5}}
文案提示禁用： {{each list as item}} {{/each}}
图标提示： {{each list as item}} {{/each}} 当前选中：{{=selected5}}
图标提示禁用： {{each list as item}} {{/each}}
HTML Code
{{!text1}}  
{{each list as item}}
<mx-radio class="mr30"
    disabled="{{=item.disabled}}" 
    name="{{=viewId}}_radioes" 
    value="{{=item.value}}" 
    text="{{=item.text}}"
    tip="{{=item.tip}}"
    tag-content="{{=item.tag}}"
    img="{{=item.img}}"
    {{:selected{refresh:true}}} />
{{/each}}
