使用说明：
类textarea，无可选下拉列表，用户输入关键词，回车确认输入结果
历史已选中的传入完整的items:array[object]
示例说明
输入值仅需要文案，无历史选中
标签至少选择一个，最多选择4个，标签文案最多2个字
配置dynamic-enter="true"表示回车生成标签
dynamic-enter-rules配置标签输入校验规则，进需要文案
当前选中
{{each items1 as item index}} {{if (index > 0)}}，{{/if}}{{=item.value}} {{/each}}
仅需文案
示例说明
输入值异步生成，有历史选中
标签必选无上限，标签文案不限制
items完整的对象数组
dynamic-enter-fn异步生成标签方法，返回对应text和value
当前选中
{{each items2 as item index}} {{if (index > 0)}}，{{/if}}{{=item.value}} {{/each}}
异步生成

JS Code

import Magix from 'magix';
export default Magix.View.extend({
    tmpl: '@index.html',
    render() {
        this.updater.digest({
            items1: [],
            items2: [{
                text: 'one',
                value: 1
            }, {
                text: 'two',
                value: 2
            }],
            dynamicEnterFn: (val) => {
                let { items2 } = this.updater.get();
                return new Promise((resolve, reject) => {
                    if (Math.ceil(Math.random() * 10) % 2 == 0) {
                        // 添加失败返回失败提示
                        reject('添加标签失败');
                    } else {
                        // 添加成功返回对象
                        resolve({
                            text: val,
                            value: items2.length + 1,
                        });
                    }
                })
            }
        });
    }
});
