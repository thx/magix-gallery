使用说明：
类textarea，无可选下拉列表，用户输入关键词，回车确认输入结果
历史已选中的传入完整的items:array[object]
实际生成值只需要文案即可
以下示例：
输入值仅需要文案，无历史选中
标签至少选择一个，最多选择4个，标签文案最多2个字
当前选中： {{each items1 as item last}} {{=item.value}}{{if !last}}，{{/if}} {{/each}}
以下示例：
输入值异步生成，有历史选中
标签必选无上限，标签文案不限制
当前选中： {{each items2 as item last}} {{=item.value}}{{if !last}}，{{/if}} {{/each}}
HTML Code
{{!text1}}  
<!-- 仅需文案 -->
<mx-taginput class="w280"
    max="4"  
    min-height="120"
    max-height="388"
    placeholder="可输入多个标签，回车分割，仅需文案"
    dynamic-enter="true"
    dynamic-enter-rules="{{@{required:true,maxlength:2}}}"
    items="{{:items1{required:true,refresh:true}}}" />


<!-- 异步生成 -->
<mx-taginput class="w280"
    min-height="120"
    max-height="388"
    placeholder="可输入多个标签，回车分割"
    dynamic-enter="true"
    dynamic-enter-fn="{{@dynamicEnterFn}}"
    dynamic-enter-rules="{{@{required:true}}}"
    items="{{:items2{required:true,refresh:true}}}" />
JS Code
{{!text2}}  
import Magix from 'magix';

export default Magix.View.extend({
    tmpl: '@index.html',
    render() {
        this.updater.digest({
            items1: [],
            items2: [{
                text: 'one',
                value: 1
            }, {
                text: 'two',
                value: 2
            }],
            dynamicEnterFn: (val) => {
                let { items2 } = this.updater.get();
                return new Promise(resolve => {
                    resolve({
                        text: val,
                        value: items2.length + 1,
                    });
                })
            }
        });
    },
});
