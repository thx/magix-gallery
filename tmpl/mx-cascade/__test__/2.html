使用场景：
配置empty-text，会在选择项中默认添加一个文案为empty-text，值为空的选项
不配置empty-text时，未选中显示请选择，选项中不添加空选项
配置empty-text="全部地域"
当前选中值：{{=selected1}}
未配置empty-text
当前选中值：{{=selected2}}
HTML Code
{{!text1}}  
<!-- 配置empty-text -->
<mx-cascade empty-text="全部地域" selected="{{:selected1{refresh:true}}}" list="{{@list}}" />

<!-- 未配置empty-text -->
<mx-cascade selected="{{:selected2{refresh:true}}}" list="{{@list}}" />
