场景1：
默认情况下（默认leaf-only=true），仅可选叶子节点
selected为叶子节点的value值
当前选中：{{=selected1}}
场景2：
可设置leaf-only=false，配置为任何一个节点都可选择
selected为选中节点的value值
当前选中：{{=selected2}}
HTML Code
{{!text1}}  
<!-- 只可选择叶子节点 -->
<mx-cascade class="w200"
    list="{{@list}}" 
    selected="{{:selected1{refresh:true}}}" />

<!-- 任意节点可选择 -->
<mx-cascade class="w200"
    leaf-only="false"
    list="{{@list}}" 
    selected="{{:selected2{refresh:true}}}" />
